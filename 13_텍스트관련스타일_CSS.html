텍스트 관련 스타일(text~)

color : 텍스트의 색을 지정하는 스타일

        선택자{
            color : 색상명/16진수/RGB/RGBA.HSL..
        }

text-decoration : 텍스트의 줄을 긋거나 줄을 없앨 때 사용

        선택자 {
            text-decoration : none/underline/overline/line-through
        }
  • underline : 텍스트 영역 아래에 줄 긋기
  • line-through : 텍스트 영역 중간에 줄 긋기
  • overline : 텍스트 영역 위에 줄 긋기
네이버로 이동

text-transform : 영문 텍스트의 대소문자 변환 시 사용

        선택자{
            text-transform : uppercase,lowercase/capitalize;
        }
  • uppercase : 모든 영문자를 대문자로 변경
  • lowercase : 모든 영문자를 소문자로 변경
  • capitalize: 영문자의 첫글자만 대문자로

text-align : 텍스트를 정렬할 때 사용

        선택자 {
            text-align : left(기본값)/justify/right/center;
        }

양쪽 정렬

피가 기관과 뜨거운지라, 쓸쓸하랴? 이성은 풍부하게 옷을 풀이 때에, 앞이 운다. 일월과 설산에서 기관과 인간은 영원히 넣는 남는 것이다. 오아이스도 그들에게 품었기 이것이다. 것이다.보라, 인류의 품에 이것이다. 그들의 날카로우나 우리는 새 이 하는 이상 보이는 노래하며 있는가? 없으면, 소금이라 그들의 있는 철환하였는가? 일월과 옷을 이상의 살았으며, 동력은 할지니, 얼음이 부패를 날카로우나 쓸쓸하랴? 뛰노는 방황하여도, 이는 있는 가지에 어디 위하여서. 그들의 얼마나 이상은 대중을 인생에 생생하며, 곳으로 듣는다.

오른쪽 정렬

청춘에서만 무엇이 석가는 꽃이 풍부하게 열매를 것이다. 용감하고 웅대한 얼음과 착목한는 열매를 바로 몸이 이상 속에 있으랴? 이상의 살 뼈 역사를 생명을 듣기만 얼마나 행복스럽고 천하를 그리하였는가? 노년에게서 얼마나 대한 얼음이 위하여, 말이다. 피어나는 굳세게 같은 찬미를 피고, 군영과 그들은 품으며, 가장 것이다. 놀이 이 피어나는 끓는 싸인 대중을 듣는다. 찾아다녀도, 피가 노년에게서 새가 넣는 곳이 사막이다. 위하여, 싶이 그들은 풀이 간에 이것이다. 돋고, 못할 긴지라 현저하게 뛰노는 목숨을 유소년에게서 것이다. 이 전인 따뜻한 노년에게서 생생하며, 가장 무한한 장식하는 그리하였는가? 피가 싶이 어디 피어나는 천하를 맺어, 소금이라 위하여서.

가운데 정렬

이상을 이는 앞이 하여도 있으랴? 뭇 살았으며, 자신과 바이며, 얼음과 얼음 같은 있는가? 꾸며 군영과 천지는 힘차게 무엇을 이것이야말로 풍부하게 보라. 열락의 어디 피고 싸인 있으랴? 만물은 이것은 따뜻한 어디 우리 생생하며, 것이다. 풀이 이상의 있는 무엇을 위하여서 따뜻한 있으며, 크고 굳세게 있는가? 우리는 이상 피어나는 그들은 싸인 이 있으랴? 튼튼하며, 그들의 얼마나 과실이 속에 용기가 그들에게 청춘 별과 아니다. 사랑의 꽃 얼마나 품었기 무한한 품으며, 용감하고 대중을 인도하겠다는 아름다우냐?

text-shadow : 텍스트에 그림자효과를 줄 때 사용

        선택자 {
            text-shadow : 가로거리(x) 세로거리(y) 번짐정도 색상;
        }

        - 번짐정도와 색상은 생략 가능
        - 가로거리와 세로거리가 양수일 경우 오른쪽 아래에 그림자 생성
        - 가로거리와 세로거리가 음수일 경우 왼쪽 위에 그림자 생성
        - 번짐정도가 0일 경우 그림자 색이 진하다
        - 색상은 기본값이 글자색과 같다
